# 🚀 Quick Start - Reflectly Mobile App

## הדרך המהירה ביותר להריץ את האפליקציה

### 📋 דרישות
- Node.js 18+ ([הורד כאן](https://nodejs.org/))
- Expo Go על הטלפון ([iOS](https://apps.apple.com/app/expo-go/id982107779) | [Android](https://play.google.com/store/apps/details?id=host.exp.exponent))

---

## ⚡ 3 צעדים פשוטים

### 1️⃣ הורד את התיקייה
```bash
# אופציה א': Clone מ-GitHub
git clone <repo-url>
cd workspace/apps/mobile

# אופציה ב': הורד רק את apps/mobile מ-Replit
# (פשוט העתק את התיקייה)
```

### 2️⃣ התקן והרץ
```bash
npm install
npm start
```

### 3️⃣ סרוק QR Code
- **iPhone:** פתח אפליקציית Camera → סרוק → הקש על ההתראה
- **Android:** פתח Expo Go → Scan QR Code

## ✅ זהו! האפליקציה רצה

### מה עכשיו?
- 📱 בדוק את המסכים השונים  
- 🎨 שנה צבעים ב-`src/theme/theme.js`
- 🎥 התחל להוסיף תכונות

---

## 🐛 בעיות?

**QR Code לא עובד?**
```bash
npx expo start --tunnel
```

**"Cannot find module"?**
```bash
rm -rf node_modules
npm install
npm start --clear
```
