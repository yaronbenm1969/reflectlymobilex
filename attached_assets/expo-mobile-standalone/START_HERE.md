# 🚀 התחל כאן - Reflectly Mobile

## מה יש בתיקייה הזאת?

זוהי **אפליקציית Expo מוכנה לשימוש** שניתן להעתיק ל-Replit חדש ולהריץ עם QR code!

---

## 📱 3 צעדים להפעלה

### 1️⃣ צור Repl חדש
- עבור ל-[replit.com](https://replit.com)
- לחץ **Create Repl**
- בחר **Blank Repl** ושם את הפרויקט `reflectly-mobile`

### 2️⃣ העלה את כל הקבצים
גרור את **כל התוכן** מהתיקייה `expo-mobile-standalone/` ל-Repl החדש

**חובה להעתיק:**
```
✅ .replit
✅ replit.nix
✅ babel.config.js    ← חשוב! ללא זה לא יעבוד
✅ package.json
✅ app.json
✅ App.js
✅ metro.config.js
✅ src/
✅ assets/
```

### 3️⃣ הרץ וסרוק QR
1. לחץ **Run** ▶️
2. המתן להתקנה (1-2 דקות)
3. **סרוק את ה-QR Code** שמופיע ב-Console
4. פתח ב-Expo Go על הטלפון! 🎉

---

## 📱 הורד Expo Go

לפני שתתחיל, הורד את האפליקציה על הטלפון:

## 🐛 בעיות נפוצות

**QR Code לא מופיע?**
- וודא ש-`babel.config.js` הועתק
- בדוק שההתקנה הסתיימה ללא שגיאות

**"Cannot connect to Metro"?**
```bash
npm start -- --reset-cache
```

**שינויים לא מופיעים?**
- שמור את הקובץ (Ctrl+S)
- נער את הטלפון → Reload

---

## 💡 טיפ חשוב

אם אתה רוצה להריץ **מקומית** במקום ב-Replit:

```bash
cd expo-mobile-standalone
npm install
npm start
```
